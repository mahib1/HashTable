# HashTable
Implementing a hashTable from scratch in C++

## What to include - 
    - A data structure with key value pairs and a table 
    - Main operation - Insertion, deletion, lookup
    - Insertion - O(1) 
    - Deletion - O(1)
    - Lookup - O(log(n)) worst case
